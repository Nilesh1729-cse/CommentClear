What Are Code Comments?

Comments are annotations embedded in source code, ignored by compilers and interpreters. They help humans understand the why behind code decisions.

There are two main types found in most languages:

//
Line Comments

Start with a symbol (like //) and comment everything to the end of the line.

/* */
Block Comments

Wrapped between start and end delimiters — can span multiple lines.

When to Remove Comments

✗ Redundant 
// increment counter
counter++;
✓ Useful 
// Debounce prevents duplicate
// API calls while typing
const search = debounce(fn, 300);
✗ Misleading 
// Parse XML response
const d = JSON.parse(res);
